# Space Mine Game 

## Overview
Space Mine Game is a Java-based arcade-style survival game where the player navigates through space while avoiding randomly spawning mines. As the game progresses, the difficulty increases with more frequent mine spawns, challenging the player's reaction time and movement skills.

This project was developed as part of a Java programming course.

---

## Gameplay
- Control your ship and avoid incoming space mines
- Mines spawn more frequently the longer you survive
- Your score increases based on survival time
- The goal is simple: survive as long as possible and achieve a high score

---

## Features
- Increasing difficulty over time
- Real-time movement and collision detection
- Score tracking system
- Simple and responsive controls

---

## Technologies Used
- Java
- Object-Oriented Programming (OOP)
- Basic game loop and event handling
